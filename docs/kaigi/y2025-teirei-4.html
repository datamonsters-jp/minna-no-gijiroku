この日の議題

会期の決定

今回の定例会の会期を、11月28日から12月16日までの19日間とすることが全員の賛成で決まりました。この期間に様々な議案が審議されます。

諸般の報告

岩国基地での空母艦載機着陸訓練（FCLP）に対して国や米軍へ繰り返し要請活動を行ったことが報告されました。また、荒川議長が全国離島振興市町村議会議長会の会長に就任したこと、岩国基地問題議員連盟の行政視察などについても報告されました。

行政報告（町長報告4件）

①岩国基地でのFCLP実施に対して防衛大臣・外務大臣などへ直接要請した経緯、②ハワイ州カウアイ郡を訪問し日系人との交流を深めたこと、③笠佐島での外国人による土地取得問題に関し住民から5,648名の署名陳情を受けたこと、④特定空家の除却費用をめぐる訴訟について相続人から全額支払いがあったため取り下げたこと、の4件が報告されました。

固定資産評価審査委員会委員の選任（同意第1号〜第4号）

任期満了（12月15日）に伴い、固定資産評価審査委員会の委員4名（伊藤和也氏・中原貞義氏・中田兼歳氏・吉村昭夫氏）を再任することについて議会の同意を求めるものです。固定資産評価審査委員会は、固定資産税の評価に不服がある住民からの申立てを審査する機関です。4名とも全会一致で同意されました。任期は令和7年12月16日から令和10年12月15日までの3年間です。

令和7年度一般会計補正予算（第4号）（議案第1号）

今年度の町の一般会計予算に約2億438万円を追加し、総額を約162億2,510万円とする補正予算です。主な内容は、米軍基地関連の特別交付金の活用（基金積立など）、職員給与の人事委員会勧告対応、大島大橋開通50周年記念映像制作費（約107万円）、産後ケア事業の拡充、広域水道のPFAS検査費用などです。討論・採決は会期最終日（12月16日）に行われます。

各特別会計補正予算（議案第2号〜第6号）

国民健康保険・後期高齢者医療・介護保険・渡船事業・下水道事業の5つの特別会計について補正予算が提案されました。主に職員人件費の給与改定への対応が中心です。渡船事業では、情島の浮桟橋更新工事において固定チェーンの腐食や物価高騰が判明したため工事費を約1,364万円増額し5,000万円とします。討論・採決は会期最終日に予定されています。

こども誰でも通園制度の条例制定（議案第7号）

令和8年4月から全国一斉に始まる「こども誰でも通園制度」（乳児等通園支援事業）の運営基準を定める条例を新たに制定するものです。生後6か月から3歳未満の子どもが、保護者が働いているかどうかに関係なく、月10時間まで1時間300円で保育施設を利用できます。町内の対象者は約100名で、すでに保育園に通う70名を除いた約30名が新たな利用対象となります。町内8か所の保育園のうち4か所が受け入れ意向を示しており、民生常任委員会に付託され採決は最終日の予定です。

選挙運動公費負担条例の改正（議案第8号）

選挙の際に候補者が使うビラやポスターの作成費を町が負担する制度について、国の基準改正に合わせて上限額を引き上げる改正です。ビラは1枚7円73銭から8円38銭に、ポスターは1枚541円31銭から586円88銭に改正されます。討論・採決は最終日の予定です。

職員給与条例の改正（議案第9号）

山口県人事委員会の勧告に基づき、町の一般職員の給料を平均3.39%引き上げるとともに、ボーナス（期末・勤勉手当）の支給月数も0.05か月分引き上げる改正です。年間の影響額は約4,593万円の見込みです。令和7年4月1日に遡って適用されます。討論・採決は最終日の予定です。

督促手数料の廃止（議案第10号）

税金や保険料などを期限までに納めなかった方へ督促状を送る際に徴収していた手数料（1通100円）を廃止する条例改正です。QRコードやスマートフォン決済の普及により手数料の仕組みが住民にとって不便になっているためで、廃止により納付書をコンビニ等で使える期間が延長され利便性が向上します。令和8年4月1日から施行予定です。

保育事業関連条例の改正（議案第11号・第12号）

国の法律や運営基準が改正されたことに合わせ、家庭的保育事業や認定こども園・保育施設の運営基準に関する条例を整備するものです。主な内容は、児童虐待禁止規定の法令条番号の修正や、健康診断の実施に関するルールの見直しです。討論・採決は最終日の予定です。

陸上競技場・総合体育館の指定管理者の指定（議案第13号）

町の陸上競技場と総合体育館を株式会社三宅商事に5年間委託する議案について質疑が行われました。施設の利用料金収入は約250万円に自動販売機等の収入を加えた額で計画されており、選定委員会は過去3年分の決算報告書から同社が黒字経営であることを確認しています。議案説明資料に掲載する情報の充実を求める意見も出されました。採決は最終日（12月16日）の予定です。

ながうらスポーツ滞在型施設の指定管理者の指定（議案第14号）

ながうらスポーツ施設等の指定管理者を瀬戸内海リゾート株式会社に1年間延長する議案について質疑が行われました。過去3年間の利用件数は令和4年度が約6万1,500件、令和5年度が約7万1,300件、令和6年度が約6万7,100件でした。審査の採点は600点満点中461点（約77%）で、財政状況の悪さが主な減点要因です。住民からの問い合わせに長期間回答しない対応問題も指摘され、町は事業者への指導を約束しました。採決は最終日の予定です。

竜崎温泉潮風の湯の指定管理者の指定（議案第15号）

竜崎温泉潮風の湯を有限会社千鳥に3年間委託する議案について質疑が行われました。事業計画書に「経費節減よりも売上増・集客増を基本とする」と記載されている点について議員から公共施設として経費削減にも取り組むべきとの指摘があり、町は事業者と改めて協議する意向を示しました。排水処理については合併浄化槽を使用していると説明されましたが、詳細な排水経路については別途図面で説明するよう求められました。採決は最終日の予定です。

サン・スポーツランド片添等の指定管理者の指定（議案第16号）

テニスコートや温浴施設、キャンプ場などをまとめて一般社団法人東和ふるさとセンターに1年間委託する議案について質疑が行われました。事業計画書に経費削減に関する具体的な記載がないことへの懸念が示され、町は口頭では削減を求めているが計画書への明記が必要との認識を示しました。また、非公募での一括委託の意図が不透明であり、今後は仕様書等の資料を議案に添付するよう求める意見も出されました。採決は最終日の予定です。

この日の議題

一般質問

この日は7名の議員が一般質問を行いました。主なテーマは、病院事業局による約2億8,000万円の国債売却損失問題、農地法の手続きを回避した非農地認定の適正運用、大島大橋の管理と防災対策、RSウイルス感染症への対応とワクチン助成、上関中間貯蔵施設建設計画、新型コロナ後の入院・入所者の面会制限、行政改革・機構改革と財政健全化などです。この日は議案の採決は行われず、一般質問のみが日程として組まれていました。

一般質問

田 田中豊文
非農地認定の適正運用について

質問

本来は農地法の手続きが必要な開発行為に対して、「非農地認定」という手続きが脱法的に使われた疑いのある事例（建設残土置き場の計画を把握しながら非農地認定した件）を取り上げ、農業委員会の対応と今後の是正を求めました。農業委員会が正式に議決したのではなく委員3名の判断で処理されている制度上の問題点も指摘しました。

答弁

農業委員会事務局長は、開発行為が関わる場合は農地法第5条の申請が本来必要であると認め、前回の答弁に誤りがあったことを謝罪しました。今後は申請書の記載内容の精度向上や農業委員会内での協議を通じ、適正な制度運用を徹底していく方針を示しました。

田 田中豊文
病院事業局の国債売却損失（約2億8,000万円）について

質問

令和2・3年度に購入した額面15億円の利付国債を令和6年度に12億円で売却し、約2億8,000万円の損失が生じた件について、購入時の意思決定の経緯と会計書類の存否、さらに責任の所在を問いました。会計規程が定める文書決裁が行われず口頭で処理されていた点や、書類が不存在のまま開示請求に応じられなかった点を強く問題視しました。

答弁

病院事業管理者は、当時は資金に余裕があり運用収入確保のために購入したが、その後の患者減少・物価高騰で資金ショートが迫り損失を承知のうえで売却が必要だったと説明しました。書類が一部整っていないことは陳謝し、過去に国債運用で19億円の利益があった点にも触れました。町長は病院と一般会計が連携して経営状況を共有しながら第2期再編を進めていく姿勢を示しました。

久 久保雅己
防災・減災対策（大島大橋の管理・避難備品・港の整備）について

質問

島民の命綱である大島大橋について県との情報共有・連携強化を求め、避難場所の備品（テントなど）の補充計画を具体的に示すよう求めました。また、橋が通行不能になった際に食料・燃料を補給できる港の整備（開作港の活用など）を繰り返し要望しました。

答弁

町長は大島大橋について今後も県・国へ適切な補修・維持管理を要望し、重量制限表示板の改善も柳井土木建築事務所に求めると答えました。避難備品は毎年予算化して更新しているとしました。港の整備は町単独では困難として、自衛隊揚陸艦も含めた多方面との連携体制構築を進めると回答しました。

山 山中正樹
RSウイルス感染症への対応とワクチン接種費用の助成について

質問

乳幼児や既往症のある高齢者が重症化しやすいRSウイルス感染症について、町としての今後の対応策と、妊婦・高齢者へのワクチン接種費用の一部助成を検討しているかを質問しました。認知度が7%と非常に低いことも踏まえて周知・支援の必要性を訴えました。

答弁

令和8年4月から妊婦向けRSウイルスワクチンが全額公費負担の定期接種となる予定ですが、60歳以上への助成は山口県内全市町で未実施のため、国・県の動向を注視するとしました。東和・大島両病院では令和7年11月末時点で計27件の接種実績があり、費用は1回2万2,275円とのことでした。

山 山中正樹
家庭用防犯カメラ設置への補助制度について

質問

犯罪抑止や安心・安全なまちづくりのために、家庭への防犯カメラ設置を支援する補助制度の創設を求めました。住民から実際に届いた相談事例（敷地への不審な侵入や嫌がらせ被害で証拠が残せない）を紹介し、導入の必要性を訴えました。

答弁

町は令和4年度から公共施設14か所に防犯カメラを設置してきたと説明しました。家庭向け補助については財源確保・ニーズ調査など検討すべき課題が多いとしながら、県内外の取組事例（下関市・和木町など）を調査して協議・検討していく意向を示しました。

山 山根耕治
住民監査請求に付された監査委員の意見への対応について

質問

住民監査請求に対して監査委員が示した3点の意見（①複数業者からの参考見積もり徴収、②システム更新の計画的準備、③部署間連携による技術支援体制の構築）について、町の受け止めと具体的な対応状況を質問しました。一部担当者への過重負担が根本原因ではないかとも指摘しました。

答弁

町長は3点すべてについて意見を真摯に受け止め、見積もりの複数者依頼徹底・計画的事務準備・部署間連携強化を図ると答えました。財務課では担当者が一人で悩まないよう早期相談を促し、プロポーザル方式ガイドラインの作成も進めているとしました。

山 山根耕治
カスタマーハラスメント対策について

質問

答弁

職員向け対応マニュアルの整備や、プライバシー保護のためひらがな表記・苗字のみの名札への変更を検討中と回答しました。国のシンポジウムにオンライン参加するなど情報収集を進めており、インターネット関連のハラスメント定義については国の指針が示され次第対応すると述べました。

占 占部智子
上関中間貯蔵施設建設計画について

質問

上関町への使用済み核燃料の中間貯蔵施設建設について、中国電力自身が「100%安全ではない」と認めているとして、事故が起きれば町民がふるさとや健康を失いかねないと指摘しました。柳井市議選で反対派9人が当選した結果の受け止めや、1市3町の直近の話し合い内容についても質問しました。

答弁

現時点では事業計画が未作成で施設詳細が不明なため賛否の見解を示す段階にないとしながら、国や事業者への説明責任を求め、1市3町で連携対応すると答えました。柳井市議選については柳井市のまちづくりの問題としつつ、1市3町での情報共有を継続していると説明しました。

占 占部智子

答弁

令和5年4月から「危険空家等除去事業」（解体費用の3分の1、上限30万円補助）を実施し、令和5年度14件・令和6年度20件の実績があると説明しました。現行計画は令和8年度までのため、令和8年度中に次期計画を見直し、さらなる施策充実に取り組む方針を示しました。

占 占部智子

占 占部智子
病院事業第2期再編計画の改訂素案（橘医院歯科の1階移転）について

質問

病院事業局が大きな赤字を抱え現金が枯渇しつつある状況で、橘医院歯科を1階へ移転するなどの費用がかかる計画を見直し、現状のままにすべきではないかと主張しました。

答弁

歯科を1階に移転することで光熱水費・修繕費の削減、受付業務の一本化による少人数体制の構築、患者が3階まで移動する負担の解消による利便性向上が図れるため、移転は必要であると考えると答えました。

小 小田貞利

質問

病院事業の財政悪化（令和10年予定だった危機が令和7年段階でマイナス5億数千万円に前倒し）と、病院・下水道・上水道への繰入増加による町財政破綻の危機を指摘しました。部課長制度の見直し、企画・立案部署の設置、行政財産の一括管理、DX推進など大胆な機構改革について、第4次行政改革大綱の計画期間中に掲げられた組織見直しが十分に進んでいないとして、具体的な改革の早急な実行を求めました。また職員が業務改善を提案できる仕組みの活用や、若手職員の意見反映についても確認しました。

町長は、物価・人件費の高騰により財政調整基金の取崩しが加速していることを認め、4つの庁舎の集約や出張所機能の郵便局への移管を住民理解を得ながら進める方針を示しました。総務部長は、空家定住対策課や地域交通課の新設など新たな課題への対応として機構改革を実施してきたと説明し、若手職員の意見交換の場を毎月開催していることも報告しました。
